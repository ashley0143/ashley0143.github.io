My issue with flatpaks is, the main flathub repo (by default) has non-free software on it. i do use flatpak on my machine, but i think traditional packaging is still suitable. i wont use a repo that has non-free software on it. i wont recommend promoting flathub, but you can use the fedora GNU/Linux repo, which i heard doesnt have non free software on it.
